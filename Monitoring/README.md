Проверка в веб-интерфейсе Prometheus
1. Перейдите в веб-интерфейс Prometheus по адресу http://localhost:9090.
2. Введите запрос probe_success в поле поиска и нажмите "Execute". Этот запрос покажет, успешно ли проходят проверки.
3. Если результатом запроса является 1, то это значит, что проверка прошла успешно. Если 0, то произошла ошибка.
 
Выполните запросы для метрик, введите следующие запросы в поле поиска и нажмите "Execute":
- Для статуса HTTP: probe_http_status_code
- Для времени выполнения проверки: probe_duration_seconds
